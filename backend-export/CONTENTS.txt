QUICK START
-----------

1. Copy harmonize.ts to your Next.js project:
   mkdir -p app/api/harmonize
   cp harmonize.ts app/api/harmonize/route.ts

2. Install dependencies:
   npm install @xmldom/xmldom

3. Start your server:
   npm run dev

4. Test the API:
   POST /api/harmonize
   - Upload a MusicXML file
   - Specify instruments (e.g., "Violin,Viola,Cello")
   - Receive harmonized output
